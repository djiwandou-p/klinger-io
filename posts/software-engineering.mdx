### Software Engineering Project.
Setiap kelompok (berisi 6-7 mahasiswa/i) memilih untuk mengerjakan software project report untuk 1 project berikut:

#### Book Rental System (Sistem Peminjaman Buku)

Terdapat 2 jenis sistem peminjaman buku: (pilih salah satu jenis saja ya)
* Fisik: meminjamkan buku secara fisik 
* E-book: meminjamkan buku secara digital
Spesifikasinya: [Book Rental System](https://docs.google.com/document/d/1V20sOcGrThNWb6wFx2COeMhxbKyMX9sq/edit?usp=sharing&ouid=117813092338862245760&rtpof=true&sd=true)

#### House/Room Rental System (Sistem Penyewaan Rumah/Kosan)

Sistem ini akan membantu Bapak/Ibu Kos untuk mengatur penyewaan rumah atau kamar yang dimiliki
Spesifikasinya: [Property Rental System](https://docs.google.com/document/d/1SumSxZZHEdDQcWVEwDsRR2KWW1wDJzPu/edit?usp=sharing&ouid=117813092338862245760&rtpof=true&sd=true)

#### Parking Booking System (Sistem Pemesanan Tempat Parkir)

Sistem ini akan membantu pengguna untuk memesan tempat parkir secara online
Spesifikasinya: [Parking Booking System](https://docs.google.com/document/d/1P5mHzSdzFtduMw7SDE4EkFlnbPtO8LXM/edit?usp=sharing&ouid=117813092338862245760&rtpof=true&sd=true)
